## Get started

1. Install dependencies

   ```bash
   npm install
   ```

   ```bash
   cd ios
   pod install
   ```

2. Start the app

   ```bash
    react-native start
   ```
   
3. Common errors

   ```bash
    NO bundle URL present
   ```
that means your metro is not running and you need to run the following command to start it:
```bash
        react-native start
   ```
